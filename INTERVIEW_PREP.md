## The cultural argument (research-backed)

Hailey opens the interview with **the diagnosis** — naming the hidden-effort problem with a 1961 study to prove it's not a Pieper-specific quirk.

**Key citations she should know cold:**

1. **James Coleman, *The Adolescent Society* (1961)** — "In every American high school, kids could be smart, but only if it looked effortless." Coleman's foundational study established that academic effort gets punished while athletic effort gets rewarded. It's a structural problem, not a character problem.

2. **John Bishop (Cornell, 2003)** — Surveyed 100,000+ students across 350 schools. Documented the **harassment curve**: kids who try hardest get teased almost as much as kids who try least. The middle gets enforced from both sides.

3. **Pluralistic ignorance (Prentice & Miller, replicated)** — every kid privately wants to be good, each one assumes nobody else does, so they all hide. The norm is *fake*. It's everyone simultaneously pretending. **This is the specific name for Pieper's problem.** Naming it out loud breaks the spell.

4. **Norman Feather** — "Tall poppy syndrome" research. High achievers face *more* peer punishment for identical mistakes than average achievers. Director's Awards often *worsen* the dynamic by spotlighting individuals.

5. **Daphna Oyserman (USC)** — Identity-based motivation. When effort feels like "what *we* do" instead of "what makes me weird," students sustain it. Effort has to be a group identity.

6. **Perkins & Berkowitz (Cornell, 1986)** — Social norms marketing. The mechanism: **publish the actual hidden norm**, behavior follows. Originally built to reduce college drinking. Mechanism has been replicated across many domains. **Woodshed's directory + pulse widget is literally this intervention applied to band.**

7. **Carol Dweck** — Effort praise (specific, strategy-tied) sustains harder-challenge-seeking; talent praise undermines it.

## The 6A stakes argument

- **Pieper enrollment 2,253**, above the new 6A floor of 2,215 (UIL Dec 2025 data). The 2026-27 marching season is Pieper's first as 6A.
- **Reagan (NEISD) finished 5th at 6A state in 2025.** That's the area benchmark Pieper now lives next to.
- **Vandegrift case study**: 4A state champs in 2013 → 6A state champs in 2022 and 2023. Director Mike Howard credits **leadership density** over talent: ~50 student leadership positions, year-round SASI curriculum (not just a May leadership camp), and the principle "the only thing you control is how hard you work." Howard's quote that lands: **"Leadership is a skill, not a talent — it can constantly improve."**
- **Hebron**: 350+ marchers, ~5 drum majors, 30+ student leaders. Density scales programs across 5A→6A, not raw numbers.
- **TMEA All-State data**: ~97% of Texas All-State band members report taking private lessons. (Per Jenny Maclay's analysis of TMEA-published numbers.) If Pieper wants more All-State chairs in 6A, normalizing private lessons is half the fight — and Woodshed's directory makes that visible.

## Visual design

**Aesthetic**: warm conservatory letterpress — Blue Note jazz program book × magazine editorial. Cream paper background, ink-dark text, Pieper purple + yellow as restrained spot colors, SVG noise grain overlay everywhere, halftone textures on featured cards, letterpress offset shadows on buttons/cards.

**Typography**:
- **Display**: Fraunces (variable, italic, SOFT+WONK axes for character)
- **Body/UI**: Instrument Sans
- **Mono labels**: JetBrains Mono

Italic serifs lean into emphasis everywhere. The `font-display-italic` class with Pieper purple is the visual signature.

## Architecture (for anyone helping maintain)

- **Stack**: Next.js 15 App Router, TypeScript, Tailwind v4, Drizzle ORM over libSQL (Turso in prod, SQLite file in dev)
- **Auth**: custom invite-code signup with bcrypt passwords + jose-signed JWT cookies + DB-backed revocable sessions (so individual devices can be signed out)
- **Email**: Resend + React Email templates, no-op fallback to console + audit_log when key unset
- **Telemetry**: PostHog (server + client), hashed user IDs, no-op without key
- **Hosting**: Vercel (linked to plantsweb3 GitHub: github.com/plantsweb3/Woodshed), Turso DB
- **Cost**: ~$0/month at school scale
